# Bolt Keyword Brainstorm

> **Purpose:** This document contains ideas for possible Bolt keywords. Nothing in this file is official until it is approved and added to `specification.md`.

| Keyword  | Proposed Meaning                 | Status      | Notes                                |
| -------- | -------------------------------- | ----------- | ------------------------------------ |
| `forge`  | Create something new             | 🟡 Proposed | Good candidate for variables         |
| `spark`  | Start or create                  | 🟡 Proposed | Good candidate for functions         |
| `craft`  | Build an object or type          | 🟡 Proposed | Fits object-oriented features        |
| `prime`  | Prepare a variable               | 🟡 Proposed | Could initialize resources           |
| `shape`  | Form a value or structure        | 🟡 Proposed | May relate to data structures        |
| `mint`   | Create something fresh           | 🟡 Proposed | Could create constants or IDs        |
| `build`  | Construct something              | 🟡 Proposed | General construction keyword         |
| `seed`   | Initialize a value               | 🟡 Proposed | Useful for random or starting values |
| `charge` | Activate or power data           | 🟡 Proposed | Could initialize systems             |
| `rise`   | Bring into existence             | 🟡 Proposed | Alternative creation keyword         |
| `cast`   | Convert or create an object      | 🟡 Proposed | May be used for type conversion      |
| `birth`  | Create a new entity              | 🟡 Proposed | Good for spawning game objects       |
| `ignite` | Start program execution          | 🟡 Proposed | Candidate for the entry point        |
| `echo`   | Output information               | 🟡 Proposed | Candidate for printing text          |
| `bloom`  | Expand or grow data              | 🟡 Proposed | Could grow arrays or collections     |
| `shift`  | Move or allocate memory          | 🟡 Proposed | Could also represent bit shifting    |
| `weave`  | Combine data together            | 🟡 Proposed | Candidate for strings or collections |
| `pulse`  | Execute every game frame         | 🟡 Proposed | Strong candidate for the game loop   |
| `flux`   | Dynamic behavior or changes      | 🟡 Proposed | Could support runtime updates        |
| `nova`   | Create something powerful or new | 🟡 Proposed | Reserved for future design           |

---

## Keyword Status

* 🟡 **Proposed** — Brainstorming stage
* 🔵 **Discussing** — Being refined
* 🟢 **Approved** — Official Bolt keyword
* ⚙️ **Implemented** — Supported by the compiler
